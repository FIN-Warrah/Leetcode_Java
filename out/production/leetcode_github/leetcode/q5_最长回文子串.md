# 5. 最长回文子串
https://leetcode-cn.com/problems/longest-palindromic-substring/

给定一个字符串 s，找到 s 中最长的回文子串。你可以假设 s 的最大长度为 1000。

## 示例
* 示例1
    ```
    输入: "babad"
    输出: "bab"
    注意: "aba" 也是一个有效答案。
    ```
* 示例2
    ```
    输入: "cbbd"
    输出: "bb"
    ```

## 解答
### 解法1：动态规划
dp[i][j] = true/false 表示从i开始到j结束的子串是否为回文串。\
状态转移方程：dp[i][j] = dp[i + 1][j - 1] && s[i] == s[j]
